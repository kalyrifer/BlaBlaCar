# Sitemap — Структура страниц

> Версия: 1.0 (MVP)

## Список страниц

| # | Страница | URL | Описание |
|---|----------|-----|----------|
| 1 | Главная | `/` | Форма поиска поездок |
| 2 | Результаты поиска | `/trips` | Список найденных поездок |
| 3 | Карточка поездки | `/trips/:id` | Детали поездки |
| 4 | Создание поездки | `/trips/new` | Форма создания поездки (водитель) |
| 5 | Мои поездки | `/my-trips` | Список моих поездок (водитель/пассажир) |
| 6 | Профиль | `/profile/:id?` | Профиль пользователя |
| 7 | Вход | `/login` | Форма входа |
| 8 | Регистрация | `/register` | Форма регистрации |
| 9 | Уведомления | `/notifications` | Список уведомлений |

---

## Навигация

```
Публичные страницы:
├── / (главная)
├── /trips (результаты поиска)
├── /trips/:id (карточка поездки)
├── /login
└── /register

Защищённые страницы (требуют авторизации):
├── /trips/new (создание поездки)
├── /my-trips
├── /profile (свой профиль)
├── /profile/:id (чужой профиль)
└── /notifications
```

## Детали страниц

### 1. Главная страница `/`

**Элементы:**
- Логотип + название приложения
- Форма поиска (город отправления, город прибытия, дата)
- Кнопка "Найти поездки"
- Ссылка "Создать поездку" (если авторизован)
- Ссылка "Войти" / "Мой профиль" (если авторизован)

**Поведение:**
- При сабмите формы → редирект на `/trips?from=...&to=...&date=...`

---

### 2. Результаты поиска `/trips`

**Элементы:**
- Хлебные крошки / форма поиска (можно изменить фильтр)
- Список карточек поездок
- Пагинация

**Поведение:**
- GET `/api/trips?from_city=...&to_city=...&date=...`
- Клик на карточку → `/trips/:id`

---

### 3. Карточка поездки `/trips/:id`

**Элементы:**
- Информация о водителе (имя, рейтинг, телефон)
- Маршрут (откуда → куда)
- Дата и время
- Цена за место
- Количество мест
- Описание
- Кнопка "Отправить заявку" (если авторизован и не водитель)
- Кнопка "Назад"

**Поведение:**
- GET `/api/trips/:id`
- POST `/api/trips/:id/requests` → модалка подтверждения

---

### 4. Создание поездки `/trips/new`

**Элементы:**
- Форма:
  - Откуда (текст)
  - Куда (текст)
  - Дата (date picker)
  - Время (time picker)
  - Места (number, 1-8)
  - Цена за место (number)
  - Описание (textarea)
- Кнопка "Создать поездку"

**Поведение:**
- POST `/api/trips`
- При успехе → редирект на `/my-trips`

---

### 5. Мои поездки `/my-trips`

**Элементы:**
- Табы: "Как водитель" / "Как пассажир"
- Список поездок с статусом заявки

**Как водитель:**
- GET `/api/trips/my/driver`
- Клик → `/trips/:id`
- Заявки на поездку (если владелец)

**Как пассажир:**
- GET `/api/requests/my`
- Статус: ожидает / подтверждена / отклонена

---

### 6. Профиль `/profile/:id?`

**Если свой профиль (`/profile`):**
- Имя, email, телефон
- Кнопка "Редактировать"
- Статистика: поездок как водитель / пассажир

**Если чужой профиль (`/profile/:id`):**
- Имя, рейтинг, статистика
- Кнопка "Написать" (будущая функция)

---

### 7. Вход `/login`

**Элементы:**
- Email (input)
- Пароль (input)
- Кнопка "Войти"
- Ссылка "Нет аккаунта? Регистрация"

**Поведение:**
- POST `/api/auth/login`
- При успехе → сохранение токена, редирект на `/`

---

### 8. Регистрация `/register`

**Элементы:**
- Email (input)
- Пароль (input)
- Имя (input)
- Телефон (input)
- Кнопка "Зарегистрироваться"
- Ссылка "Есть аккаунт? Вход"

**Поведение:**
- POST `/api/auth/register`
- При успехе → сохранение токена, редирект на `/`

---

### 9. Уведомления `/notifications`

**Элементы:**
- Список уведомлений
- Индикатор непрочитанных
- Клик → чтение + переход по related_trip_id

**Поведение:**
- GET `/api/notifications`
- PUT `/api/notifications/:id/read`

---

## Модальные окна

| Модалка | Триггер | Описание |
|---------|---------|----------|
| Заявка на поездку | Клик "Отправить заявку" | Выбор мест, сообщение |
| Подтверждение заявки (водитель) | Новое уведомление | Принять/отклонить |
| Выход | Клик "Выйти" | Подтверждение выхода |
