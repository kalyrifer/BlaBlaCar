# E2E Сценарий тестирования

## Цель
Протестировать ключевой сценарий: создать поездку → найти → отправить заявку → подтвердить.

## Участники
- **Водитель** (driver@test.com)
- **Пассажир** (passenger@test.com)

## Пошаговый сценарий

### Шаг 1: Регистрация водителя
1. Открыть http://localhost:5173/register
2. Заполнить форму:
   - Email: driver@test.com
   - Пароль: driver123
   - Имя: Иван
   - Телефон: +79001234567
3. Нажать "Зарегистрироваться"
4. Ожидаемый результат: редирект на главную, пользователь авторизован

### Шаг 2: Создание поездки водителем
1. В навигации нажать "Создать поездку"
2. Заполнить форму создания поездки:
   - Откуда: Москва
   - Куда: Санкт-Петербург
   - Дата: завтрашняя дата
   - Время: 10:00
   - Цена: 1500
   - Места: 3
   - Описание: Комфортная поездка на Mercedes
3. Нажать "Создать поездку"
4. Ожидаемый результат: поездка создана, редирект на страницу "Мои поездки"

### Шаг 3: Логаут водителя
1. Нажать "Выйти" в навигации
2. Ожидаемый результат: пользователь разлогинен

### Шаг 4: Регистрация пассажира
1. Открыть http://localhost:5173/register
2. Заполнить форму:
   - Email: passenger@test.com
   - Пароль: passenger123
   - Имя: Петр
   - Телефон: +79009876543
3. Нажать "Зарегистрироваться"
4. Ожидаемый результат: редирект на главную, пользователь авторизован

### Шаг 5: Поиск поездки пассажиром
1. На главной странице в форме поиска:
   - Откуда: Москва
   - Куда: Санкт-Петербург
   - Дата: завтрашняя дата
2. Нажать "Найти"
3. Ожидаемый результат: отображается список найденных поездок, включая созданную водителем

### Шаг 6: Просмотр деталей поездки
1. В списке найденных поездок кликнуть на поездку водителя "Иван"
2. Ожидаемый результат: открывается страница с деталями поездки

### Шаг 7: Отправка заявки пассажиром
1. На странице поездки нажать "Забронировать"
2. Ожидаемый результат: заявка отправлена, кнопка меняется на "Заявка отправлена"

### Шаг 8: Логаут пассажира
1. Нажать "Выйти" в навигации

### Шаг 9: Вход водителя
1. Открыть http://localhost:5173/login
2. Ввести credentials водителя
3. Нажать "Войти"

### Шаг 10: Просмотр заявок водителем
1. Перейти в "Мои поездки"
2. Выбрать созданную поездку
3. Ожидаемый результат: в разделе "Заявки" отображается заявка от пассажира Петра

### Шаг 11: Подтверждение заявки водителем
1. Нажать "Подтвердить" напротив заявки пассажира
2. Ожидаемый результат: заявка подтверждена, статус меняется на "Подтверждено"

### Шаг 12: Проверка уведомлений пассажиром
1. Войти как пассажир
2. Перейти в "Уведомления"
3. Ожидаемый результат: отображается уведомление о подтверждении заявки

## Тестовые данные

| Поле | Значение |
|------|----------|
| Водитель email | driver@test.com |
| Водитель пароль | driver123 |
| Пассажир email | passenger@test.com |
| Пассажир пароль | passenger123 |
| Маршрут | Москва → Санкт-Петербург |
| Дата | +1 день от текущей |
| Время | 10:00 |
| Цена | 1500 ₽ |
| Места | 3 |

## Ожидаемые статусы заявки
- `pending` — заявка отправлена, ожидает подтверждения
- `confirmed` — заявка подтверждена водителем
- `rejected` — заявка отклонена водителем
- `cancelled` — заявка отменена пассажиром

## Критерии приёмки
- [ ] Водитель может создать поездку
- [ ] Пассажир может найти поездку по маршруту и дате
- [ ] Пассажир может отправить заявку на поездку
- [ ] Водитель видит входящие заявки
- [ ] Водитель может подтвердить заявку
- [ ] Пассажир получает уведомление о подтверждении
